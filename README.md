# Aletheia's Portfolio

My personal portfolio — a glassmorphism **bento dashboard** with a profile card,
live clock, calendar, and pages for my projects and writing.

**Live:** not deployed yet

## Built with

- **Next.js 15** (App Router) + **React 19**
- Plain CSS — no framework
- **Space Grotesk** (body & headings) + *Averia Gruesa Libre* (display), via `next/font`

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
